# Бот-координатор — @botansfree_bot

Telegram-бот на aiogram 3: чисто информационное меню, одинаковое для всех
пользователей. Никакой пересылки сообщений и админки — если нужен реальный
диалог, бот просто даёт ссылку на аккаунт @botans_free.

## Как это устроено

- `/start` показывает приветствие и меню: «Частые вопросы», «Каталог ботов»,
  «Сайт», «Заказать бота».
- «Частые вопросы» — 3 готовых вопроса с ответами и кнопка со ссылкой на
  @botans_free, если ответа не нашлось.
- «Каталог ботов» — пока заглушка «Скоро будет добавлен»: цены и список
  сознательно не публикуем (см. `ЦЕНЫ.md` в корне проекта).
- «Сайт» и «Заказать бота» — обычные ссылки (сайт и чат с @botans_free),
  открывающиеся вне бота.
- Никакого хранения данных о пользователях бот не ведёт.

## Установка

```bash
cd support-bot
python3 -m venv venv
source venv/bin/activate       # Windows: venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
```

## Настройка `.env`

Откройте `.env` и впишите **BOT_TOKEN** — токен бота, который выдал
[@BotFather](https://t.me/BotFather) при создании @botansfree_bot.

`.env` не попадает в git (см. `.gitignore`) — храните токен только там.

## Запуск

```bash
python bot.py
```

Бот работает через long polling — отдельный сервер с HTTPS не нужен, можно
запускать хоть с ноутбука, хоть на любом VPS.

## Что дальше

- Наполнить «Каталог ботов» реальным содержимым, когда определитесь с ценами
  для публичного показа (см. `ЦЕНЫ.md`).
- Для продакшена на постоянном сервере можно перейти с polling на webhook.
